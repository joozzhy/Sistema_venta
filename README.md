# Sistema_venta
sistema de "ventas" donde se puede agregar el producto, asignado <ul>nombre producto</ul> <ul>descripcion producto</ul> <ul>valor producto</ul> <br> se puede agregar al carrito, ver las ventas editar la cantidad y eliminar la venta. <br> los productos se almacenan en un json el cual se carga cuando se inicia el localhost. <br> por ahora funciona en el navegador :D
